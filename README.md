## 📌 About

A fully animated personal portfolio designed to present automation and software development work. The site features a 3D robot hero that tracks the user's mouse cursor, rotating headline text, interactive project cards with hover effects, and a scrolling technology strip — all built with a dark, minimal aesthetic.

---

## ✨ Sections

**Hero**
- 3D robot model that follows the mouse in real time
- Rotating headline cycling through **Automate.** → **Scale.** → **Code.**
- CTA buttons linking to active projects and contact

**Active Projects**
- Tabbed navigation between **Automações** and **Sites & Systems**
- Cards for each project with title, description and direct link

**Stack & Technologies**
- Infinite horizontal scroll strip displaying all technologies used

**What I Deliver**
- Animated service cards: Automation, Artificial Intelligence, Software Engineering, Cloud & M365, SaaS & Web
- Hover effect brings the focused card to the foreground

**Contact**
- Terminal-style animation with call to action
- Direct links to WhatsApp, LinkedIn and GitHub

## 🚀 Running locally

**Requirement:** Node.js & npm installed

```bash
# Clone the repository
git clone https://github.com/Tharomeo/portfolio.git
cd portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:8080](http://localhost:8080) in your browser.
